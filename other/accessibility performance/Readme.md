1. ###  aria-live-explanation
  1. ***aria-live*** is an accessibility feature for screen readers.
  In simple words:
  👉 It tells a screen reader:
  “If this part of the page changes, announce it out loud.”
  2. Normally:
  Screen readers only read what was there when the page loaded
  But some content changes dynamically, like:
  notifications (“Message sent”)
  form errors (“Password is too short”)
  loading updates (“3 results found”)
